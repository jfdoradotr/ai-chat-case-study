# ai-chat — Study Case

Study-case repository built while following the [Swiftful Thinking — SwiftUI Advanced Architecture](https://www.swiftful-thinking.com/architecture) course by Nick. The app is an AI Chat client that generates avatars and lets users chat with them in real time.

## Scope

The course is split into three parts and this repo evolves alongside them:

1. **AI Chat App** — Onboarding, tab bar flow, reusable components, real-time chat, In-App Purchases, authentication, local persistence, and a remote database.
2. **MVP Essentials** — Analytics, AB testing, a logging package, a reusable purchases package, Cloud Functions, and CI/CD via Xcode Cloud.
3. **Enterprise Architecture** — Refactors through MVC, MVVM, and VIPER, modularizing the codebase with Swift Packages and a RIBs-like pattern.

## Stack

- Swift 6, Swift Concurrency, Observation framework
- SwiftUI, SwiftData, FileManager, UserDefaults
- StoreKit, RevenueCat
- Firebase (Authentication, Analytics, Crashlytics, Remote Config, Cloud Functions)
- Mixpanel
- Xcode Cloud, SwiftLint
